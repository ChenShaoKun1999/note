# 泰拉瑞亚

## 流程

1. 探索地表和洞穴、挖矿
2. 史莱姆王：史莱姆雨期间随机生成。掉落物在前期比较有用，但不是必须
3. 克苏鲁之眼：夜晚有几率生成；在夜间使用可疑眼球召唤（在恶魔祭坛 / 猩红祭坛用6个晶状体合成）
4. 世界吞噬怪 / 克苏鲁之脑：摧毁3个暗影珠 / 猩红之心召唤；在腐化之地 / 猩红之地使用蠕虫诱饵 / 血腥脊椎召唤（在祭坛用腐化 / 猩红怪物掉落物合成）。对于克苏鲁之脑建议铁以上防具、高攻速高击退武器
5. 探索地下丛林（打破蜂巢中的幼虫或者用毒刺等材料合成憎恶之蜂可以召唤蜂王）、地下沙漠、地狱、地牢（需要在晚上和地牢门口的老人对话，击败骷髅王才能进入）
6. 升级装备并且做好基础建设和物资储备（因为进入困难模式之后更难刷资源）

# Emuera

## 宏（マクロ）

保存与使用方法：shift+F1~F12保存宏，F1~F12使用宏。另外，ctrl+0~9可以切换不同的宏组（マクログループ），因此一共能保存12×10=120个宏。游戏退出后，宏保存在macro.txt

语法：`\e`相当于esc，`\n`相当于回车，`()*n`表重复。使用例：`(0\e)*3`实现重复三次选择0选项

## 调试（デバッグ）

在命令行加上`-Debug`参数启动Emuera，就能在调试窗口看各种信息

调试模式下，可以输入`@command`直接执行，比如`@PRINTV FLAG:200`（不过，作为防作弊机制，执行命令之后玩家名会被更改为イカサマ）

特殊调试指令：`REBOOT`重启，`OUTPUT`将文本输出到log，`EXIT`退出，`CONFIG`打开设置窗口，`DEBUG`打开调试窗口

Emuera使用基于Basic语言的eramaker basic语言。[语法](https://ja.osdn.net/projects/emuera/wiki/FrontPage)、[教程](https://seesaawiki.jp/eratoho/d/ERB%b9%bd%ca%b8%b9%d6%ba%c2)

# Noita

## 施法规则

noita施法类似于卡牌游戏抽牌模型。法杖内的法术处于以下三个状态之一：

- 牌堆：未被释放的法术
- 手牌：即将释放的法术
- 弃牌堆：释放过的法术（或者释放失败的、被其他法术跳过的）

点击左键施法相当于进行一次抽牌。抽牌遵循以下规则：

1. 初始的抽取数是法杖的spell per cast
2. 依次抽取法术。投射物法术消耗1次抽取；投射修正不消耗；N重施法消耗1次然后增加N个
3. 遇到了无法施放的法术（魔力不足，剩余使用次数不足），直接放入弃牌堆并多抽一次
4. 如果过程中牌堆无牌，则将弃牌堆的牌移回牌堆
5. 当没有剩余抽取数时，或者全部法术进入手牌时，结束抽取

**抽牌结束后**，将手牌中所有法术发出，并移动到弃牌堆，然后进入**施放延迟时间**（施放延迟为法杖的延迟+法术的延迟）

如果牌堆空了，或者抽卡时曾**把牌堆抽空**过（规则4），进入**充能时间**。充能的同时也在计算施放延迟，要等到两者均结束才能再次施法（即，需要等待两者中较长的一个结束）

## 触发法术

触发算作一次单独的施法，而且触发法术不计入施放延迟。比如：

```
双重施法 - 触发型火花弹 - 分裂弹 - 能量球 - 触发型能量球
```

**第一次施法**

括号中的数字为这个法术产生的剩余抽取数的变化，列表等级更高的是被触发的另一次施法

- 双重施法（1→2）
- 触发型火花弹（2→1）
  - 分裂弹（算作另一次施法，*1→0*）

- 能量球（1→0）

发出火花弹和能量球，火花弹击中物体后触发分裂弹。注意，在分裂弹触发之前它就已经离开手牌了，因此原则上可能同时触发两个分裂弹

**第二次施法**

- 触发型能量球（1→0）
  - 双重施法（另一次施法，*1→2*）
  - 触发型火花弹（*2→1*）
    - 分裂弹（更深一层的触发，1→0）
  - 能量球（*1→0*）

除了触发型能量球以外的法术都是从弃牌堆取回来的。发出一个能量球，它击中物体后触发火花弹和能量球，火花弹击中物体后再触发一个分裂弹。本次施法结束后进入充能时间

这个例子中，利用从弃牌堆取回来的法术，一轮施法每个法术释放了两次（触发型能量球除外），这种技术称作法术回绕。法术回绕能减少需要的充能次数，加快施法速度

## 机枪

链锯法术有两个效果：

1. 减少0.17秒充能时间
2. 清零施法延迟（不过，无法影响它后面法术的延迟。如果想要零延迟施法，**链锯后面不能有增加延迟的法术**）

如果让施法的充能时间和延迟都降低到0，施法速度将达到游戏允许的最高速（应该是每秒60次）
